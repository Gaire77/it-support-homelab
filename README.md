##### \# IT Support Home Lab



A hands-on Active Directory environment and IT help desk simulation built to demonstrate practical skills relevant to entry-level IT support, help desk technician, and system administrator roles. This project documents the end-to-end setup of a virtualized enterprise IT environment, including Windows Server administration, Active Directory configuration, Group Policy management, and ticket-based support workflows using osTicket.



##### \## Project Goals



\- Build a Windows Server 2022 domain controller running Active Directory, DNS, and DHCP

\- Configure realistic Group Policy Objects (GPOs) covering security, user environment, and access control

\- Deploy and configure OS Ticket as a ticket management system

\- Document and resolve realistic IT support scenarios end-to-end through the ticket lifecycle

\- Demonstrate troubleshooting, documentation, and communication skills expected in a help desk role



##### \## Lab Environment



| Component | Software |

|-----------|----------|

| Hypervisor | Oracle VirtualBox |

| Domain Controller | Windows Server 2022 (Evaluation) |

| Client Workstation | Windows 10/11 Enterprise (Evaluation) |

| Ticketing System | osTicket (open source) |

| Web Stack | XAMPP (Apache, MySQL, PHP) |

| Scripting | PowerShell |
